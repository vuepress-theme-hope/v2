Plantuml

小于 1 分钟MarkdownMarkdown图表
此页内容
  • 设置
  • 格式
  • 示例

让你站点中的 Markdown 文件支持 plantuml。

设置

你可以通过下方启用此功能:

import { hopeTheme } from "vuepress-theme-hope";

export default {
  theme: hopeTheme({
    plugins: {
      mdEnhance: {
        plantuml: true,
      },
    },
  }),
};

格式

你可以插入plantuml 支持的相同内容，例如:

@startuml
内容
@enduml

    vuepress-plugin-reading-time2

    预计阅读时间与字数统计生成

    这个插件将会向 page 对象注入预计阅读时间与字数统计。

    会自动向 page 对象注入 readingTime:

    interface ReadingTime {
  /** 分钟数 */
  minutes: number;
  /** 字数 */
  words: number;
}

    # 插件选项

    # wordPerminute

    • 类型: number
    • 默认值: 300

    每分钟阅读字数

    # locales

    interface ReadingTimeLocaleData {
  /**
   * 字数模板，模板中 `$word` 会被自动替换为字数
   */
  word: string;

  /**
   * 小于一分钟文字
   */
  less1Minute: string;

  /**
   * 时间模板
   */
  time: string;
}

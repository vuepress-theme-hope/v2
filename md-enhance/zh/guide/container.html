                          自定义容器

                          Mr.HopeFebruary 3, 2022小于 1 分钟

                          插件可以为你添加提示、注释、信息、注意、警告和详情自定义容器的支持。

                          # 配置

                          module.exports = {
  plugins: [
    [
      "md-enhance",
      {
        // 启用自定义容器
        container: true,
      },
    ],
  ],
};

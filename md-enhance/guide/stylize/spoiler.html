Spoiler

Less than 1 minute
On This Page
  • Settings
  • Syntax

Hide spoiler contents in your VuePress site.

Settings

import { mdEnhancePlugin } from "vuepress-plugin-md-enhance";

export default {
  plugins: [
    mdEnhancePlugin({
      // enable spoiler
      spoiler: true,
    }),
  ],
};

Syntax

Use !! !! to mark a content as spoiler.

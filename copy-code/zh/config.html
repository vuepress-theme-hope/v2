          # selector

          • 类型: string | string[]
          • 默认值: '.theme-default-content div[class*="language-"] pre'

          # duration

          • 类型: number
          • 默认值: 2000

          提示消息显示时间，设置为 0 会禁用提示。

          # showInMobile

          • 类型: boolean
          • 默认值: false

          是否展示在移动端

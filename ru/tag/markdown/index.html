                                                              Vue Playground

                                                              Пусть файл Markdown поддерживает playground vue на вашем сайте VuePress.

                                                              Mr.HopeMarkdownMarkdownPlaygroundОколо 1 мин
                                                              MdEnhance Plugin Config

                                                              Введение

                                                              Плагин vuepress-plugin-md-enhance включен по умолчанию и предоставляет улучшения Markdown.

                                                              vuepress-theme-hope передает plugins.mdEnhance в параметрах темы в качестве параметров плагина для плагина vuepress-plugin-md-enhance.

                                                              Mr.HopeКонфигMarkdownРазметкаКонфигурация плагинаКонфиг темыОколо 4 мин
                                                              Markdown

                                                              VuePress в основном генерирует страницы из файлов Markdown. Таким образом, вы можете легко использовать его для создания документации или блогов.

                                                              Вы должны создавать и записывать файлы Markdown, чтобы VuePress мог преобразовывать их на разные страницы в соответствии со структурой файла.

                                                              Компоненты

                                                              Используя vuepress-plugin-components, вы можете импортировать и использовать некоторые компоненты в ваших файлах Markdown.

                                                              Доступные компоненты:

                                                              • AudioPlayer
                                                              • Badge
                                                              • BiliBili
                                                              • Catalog
                                                              • CodePen
                                                              • FontIcon
                                                              • PDF
                                                              • StackBlitz
                                                              • VideoPlayer
                                                              • YouTube

                                                              По умолчанию <Badge /> и <FontIcon /> включены.

                                                              Чтобы включить компоненты, вы должны установить plugin.components.components с массивом имен компонентов.
